# Simple Interest Calculator

This project is a simple interest calculator that calculates the interest based on principal amount, rate of interest, and time.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Features
- Calculates simple interest
- Easy to use
- Beginner-friendly project

## Technologies Used
- (Example: Shell Script / Python / HTML / JavaScript)

## How to Run
1. Clone the repository
2. Run the program
3. Enter principal, rate, and time

## Example
Input:
Principal = 1000
Rate = 5
Time = 2

Output:
Simple Interest = 100
